# 한국어 향수 조합 추천 웹앱 (name2perfume) 완료 보고서

한국어 이름을 분석하여 어감과 감성에 꼭 맞는 프리미엄 향수 레시피를 제안해 주는 웹앱(MVP)을 성공적으로 구현하였습니다.

## 개발 완료 항목 요약

1. **프로젝트 설정 및 의존성 구성**
   - Vite + React + TypeScript + Tailwind CSS v3 기반 프로젝트 구조 정의
   - 프리미엄 감성의 폰트(Cinzel, Noto Sans KR, Inter) 및 Tailwind 테마(숲속 초록색 감성 `forest` 계열 및 럭셔리 크림 `luxury` 계열) 설정
   - Node.js 구버전 및 윈도우 환경 대응을 위한 전역 프로세스 레벨의 WebCrypto(crypto.randomBytes) 폴리필 구축 완료

2. **핵심 추천 및 분석 로직 구현**
   - [analyzeName.ts](file:///c:/Users/이대경/Documents/workspace/260705_name2perfume/src/logic/analyzeName.ts): 한글 유니코드 음절 분리 알고리즘을 구현하여 초성/중성/종성 정보를 추출하고, 모음/받침/글자수를 기반으로 감성 이미지 및 키워드 태그 목록을 동적으로 구성. 다양한 매칭을 위해 각 성격별 대표 태그를 교차 배치(Interleave)하여 UI 및 추천 로직에 전달
   - [recommendPerfume.ts](file:///c:/Users/이대경/Documents/workspace/260705_name2perfume/src/logic/recommendPerfume.ts): 도출된 이름 태그와 향료 태그간의 매칭율을 계산한 뒤 탑 2개, 미들 3개, 베이스 2개의 향료를 선별. 이때 이름 고유의 해시값(DJB2 Hash)을 정렬 가중치로 활용하여, 무작위성을 배제하면서도 서로 다른 이름에 대해 고유한 향수 조합이 매칭되도록 100% 결정론적(Deterministic) 타이브레이커 알고리즘 적용
   - [calculateRatio.ts](file:///c:/Users/이대경/Documents/workspace/260705_name2perfume/src/logic/calculateRatio.ts): 노트별 배분 비율(기본 Top 25%, Mid 45%, Base 30%)에 맞춰 개별 향료의 정수 비율을 분배하며, 나머지(Remainder) 분배법을 통해 비율의 총합이 항상 정확히 100%가 되도록 정밀 조정

3. **프리미엄 UI 및 향료 이름 편집 기능**
   - [App.tsx](file:///c:/Users/이대경/Documents/workspace/260705_name2perfume/src/app/App.tsx): 숲속 아틀리에 느낌을 주는 다크 그린 테마와 글라스모피즘 스타일, 세련된 아날로그 향수병 그래픽(SVG) 디자인 적용
   - 빠른 사용성 및 검증을 위해 조사 대기 시간(Simulated Loading Timeout)을 1500ms에서 300ms로 대폭 단축
   - 프리셋 설정 및 직접 조율 슬라이더를 장착하여 실시간 비율 계산 연동
   - 로컬 스토리지를 활용하여 48개 향료에 대한 커스텀 향료명(예: Top-01 -> 레몬) 관리 패널 구축

## 검증 및 수동 확인 결과

1. **이름 입력 유효성**:
   - 한글 이외의 입력(영문, 숫자 등)이나 비정상적인 길이(1자 혹은 6자 이상) 입력 시 경고 메시지가 자연스럽게 출력됨을 확인하였습니다.
2. **비율 총합 100% 검증**:
   - 추천된 탑, 미들, 베이스 각 향료의 비율을 합산하면 어떤 구성이나 커스텀 슬라이더 값에서도 정확히 100%가 도출됨을 검증하였습니다.
3. **결정론적 매칭 검증**:
   - 동일한 이름을 여러 번 입력하여 추천을 진행하여도 항상 동일한 향수 레시피 조합이 오차 없이 일관되게 반환됨을 확인하였습니다.
4. **향료명 수정 기능**:
   - 향료명 관리 패널에서 향료 이름을 변경하면 결과 카드에 즉각 반영되며, 브라우저 새로고침 후에도 유지됨을 확인하였습니다.
5. **이름별 고유성 및 Vibe 태그 차별화 검증**:
   - '김민수'와 '이영희' 등 모음 구성은 유사하지만 받침 유무가 다른 이름들에 대해 UI에 표시되는 Vibe 태그와 실제 추천 향수 조합이 각각 다르고 개성적으로 출력됨을 직접 검증하였습니다.
